
# devcloud Mail UI Mock

This is the visual and interaction mock for the `devcloud` Mail MVP dashboard.
Use it as a reference for layout, states, motion, and visual tone. The production Go dashboard should still follow `docs/design-mail-ui.md` and should not inherit this mock's React stack by default.

## Running the code

Run `npm install` to install the dependencies.

Run `npm run dev` to start the development server.

Run `npm run build` before changing the mock substantially.
